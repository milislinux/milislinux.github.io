Milis Linux 2.0

Milis Linux 2.0 close to stable but we published last image as Beta. There are two image for 2.0, Minimal and XFCE4. Milis 2.0 uses X as default graphics server and Sysvinit as init system.

  • Milis 2.0 Minimal Beta (Kasım 2020) - shasum
